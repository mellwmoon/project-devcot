# DEVCOT
*A 2nd year capstone group project*

**Devcot** is a website that introduces computer technologies' fundamentals, all the way to advanced topics with an easy-to-understand delivery, and a *ton* of knowledge across all over the field pressented to you on a silver platter - Managed and created by three Computer Engineering Students.
